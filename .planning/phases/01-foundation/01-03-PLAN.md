<task type="auto">
  <name>Task 2: Create LanguageContext with provider and hook</name>
  <files>src/contexts/LanguageContext.tsx</files>
  <action>
  Create `src/contexts/LanguageContext.tsx`:

  1. Import `Language` from `../types/content`

  2. Define `LanguageContextType`:
     - `language: Language` — current language
     - `setLanguage: (lang: Language) => void` — set specific language
     - `toggleLanguage: () => void` — toggle between fr/en

  3. Create the React Context with `createContext<LanguageContextType | undefined>(undefined)`

  4. Create `LanguageProvider` component:
     - On mount, detect initial language:
       a. Check `localStorage.getItem('altixia-language')` — if 'fr' or 'en', use it
       b. Otherwise check `navigator.language` — if starts with 'fr', use 'fr'; if starts with 'en', use 'en'
       c. Default to 'fr' if neither matches
     - Use `useState<Language>` with lazy initializer for the detection logic
     - `setLanguage` function: updates state AND writes to `localStorage.setItem('altixia-language', lang)`
     - `toggleLanguage` function: switches 'fr' to 'en' and vice versa, persists
     - Wrap children in `LanguageContext.Provider` with memoized value

  5. Create `useLanguage` hook:
     - Returns `useContext(LanguageContext)`
     - Throws error if used outside LanguageProvider

  6. Export: `LanguageProvider`, `useLanguage`

  7. Update `src/App.tsx` to wrap content with `<LanguageProvider>`:
     ```tsx
     import { LanguageProvider } from './contexts/LanguageContext'

     function App() {
       return (
         <LanguageProvider>
           <div>
             <h1 className="text-3xl font-bold">Altixia LLC</h1>
           </div>
         </LanguageProvider>
       )
     }
     ```
  </action>
