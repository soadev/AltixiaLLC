  <name>Task 1: Set up Google Fonts and shared layout</name>
  <files>vite.config.ts, src/layouts/RootLayout.tsx, index.html</files>
  <action>
  **Google Fonts Setup:**

  1. Install vite-plugin-webfont-dl:
     ```bash
     npm install -D vite-plugin-webfont-dl
     ```

  2. Update `vite.config.ts` to add the webfont plugin. Choose from the approved font pairs in CONTEXT.md. Recommended: **Playfair Display** (serif, titles) + **DM Sans** (sans-serif, body):
     ```typescript
     import { defineConfig } from 'vite'
     import react from '@vitejs/plugin-react'
     import tailwindcss from '@tailwindcss/vite'
     import webfontDownload from 'vite-plugin-webfont-dl'

     export default defineConfig({
       plugins: [
         react(),
         tailwindcss(),
         webfontDownload([
           'https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;600;700&display=swap',
           'https://fonts.googleapis.com/css2?family=DM+Sans:wght@400;500;600;700&display=swap',
         ]),
       ],
     })
     ```

  3. Update `src/index.css` to set font families via Tailwind CSS 4 theme:
     ```css
     @import "tailwindcss";

     @theme {
       --font-serif: "Playfair Display", serif;
       --font-sans: "DM Sans", sans-serif;
       --color-primary: #3877AF;
       --color-primary-light: #D5E8F8;
       --color-dark: #333A49;
       --color-medium-blue: #9BCAEF;
       --color-off-white: #F7F9FC;
       --color-light-gray: #E8EDF2;
       --color-text-secondary: #6B7280;
       --color-hover-blue: #2D6193;
       --color-success: #10B981;
     }
     ```

  **Shared Layout:**

  4. Create `src/layouts/RootLayout.tsx`:
     - Import `Outlet` from `react-router-dom`
     - Render a minimal wrapper:
       ```tsx
       import { Outlet } from 'react-router-dom'

       export function RootLayout() {
         return (
           <div className="min-h-screen font-sans text-dark">
             {/* Navbar will be added in Phase 2 */}
             <main>
               <Outlet />
             </main>
             {/* Footer will be added in Phase 4 */}
           </div>
         )
       }
       ```
     - The `font-sans` class applies DM Sans as body font via the Tailwind theme
