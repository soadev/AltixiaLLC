  <action>
1. Create `src/components/LucideIcon.tsx` — a dynamic icon resolver that maps string icon names (from content data) to Lucide React components:
   - Accept props: `name: string`, `size?: number` (default 20), `className?: string`
   - Import from `lucide-react` using dynamic lookup: `import * as LucideIcons from 'lucide-react'`
   - Look up `LucideIcons[name]` and render it, return null if not found
   - Export as named export `LucideIcon`

2. Create `src/hooks/useScrollPosition.ts` — custom hook for scroll-aware navbar:
   - Accept `threshold` parameter (default 50px)
   - Use `useState<boolean>` initialized to false
   - Use `useEffect` with passive scroll event listener on window
   - Set `isScrolled` to `true` when `window.scrollY > threshold`
   - Clean up listener on unmount
   - Return `isScrolled` boolean
  </action>
